# Marketing Funnel Analysis Dashboard

## Project Overview

This project analyzes marketing funnel performance using the Bank Marketing Dataset. The goal is to identify conversion rates, customer acquisition patterns, and opportunities to improve marketing effectiveness.

## Problem Statement

Businesses invest significant resources in marketing campaigns, but not all leads become customers. This project helps answer the following questions:

* What is the overall conversion rate?
* Which communication channels perform best?
* Which customer segments are most likely to convert?
* What factors influence customer acquisition?

## Dataset

Dataset Used: Bank Marketing Dataset

Source:
https://archive.ics.uci.edu/dataset/222/bank+marketing

The dataset contains customer demographic information, marketing campaign details, and subscription outcomes.

## Key Insights

* Cellular contact method generated the highest number of conversions.
* Certain professions showed higher conversion rates than others.
* Customer conversion varied across months.
* Demographic factors influenced subscription success.

## Business Recommendations

1. Focus marketing efforts on high-performing communication channels.
2. Target customer segments with higher conversion potential.
3. Optimize campaigns during high-performing months.
4. Improve follow-up strategies for non-converted leads.
